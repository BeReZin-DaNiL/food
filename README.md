# Инструкция по запуску сайта

## 1. База данных (PostgreSQL)

1. Установите PostgreSQL с официального сайта.
2. Создайте базу данных с именем `cwp1_231`.
3. Выполните SQL-скрипт из файла `schema.sql` в этой базе данных (через pgAdmin или консоль).

## 2. Настройка PHP

1. Убедитесь, что в вашем `php.ini` раскомментировано расширение для PostgreSQL:
   ```ini
   extension=pdo_pgsql
   ```
   (Уберите точку с запятой `;` перед началом строки).

## 3. Настройка проекта

1. Откройте файл `db.php`.
2. Убедитесь, что переменная `$pass` соответствует паролю, который вы задали при установке PostgreSQL.
   ```php
   $pass = 'ваш_пароль';
   ```

## 4. Запуск

1. Откройте терминал в папке проекта.
2. Запустите встроенный сервер PHP:
   ```bash
   php -S localhost:8000
   ```
3. Откройте браузер по ссылке: [http://localhost:8000](http://localhost:8000)
